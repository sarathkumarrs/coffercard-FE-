## Project Overview

A MVP marketing tool for SME's around the globe, this have two active campaigns, which is Spin and win, Scratch card.

## Technologies Used

* React
* Fetch, Axios, React-router-dom, roulette for spin wheel
* Node.js
* npm 

## Prerequisites

* Node.js 
* npm 
* React >= 18.2



   ```bash

   ```
